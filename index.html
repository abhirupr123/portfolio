Flux Fashion

Flux Fashion is an E-Commerce fashion website for mens' clothing made using React and using the Firestore Database for storage of product details, user information, payment details and their retrieval. Firebase Authentication is implemented for user Sign-In and new user registrations. Frontend is developed using Tailwind CSS and Chakra-UI. Redux and Redux-Toolkit is used for the Shopping Cart. Razorpay is integrated as the Payment Gateway for transactions through a backend server implemented with NodeJS and Express.
